Aufstieg the package manager

Hello, I wrote whis package when i was bored in November or December, it's still kind of unusable but it can compile downloaded source code and delete packages binaries, and now, I want to write documentation for it because I'm bored again

How to use it:

You can install packages with "aufstieg add (package name)", to check package existence, search it in /usr/ports/

You can remove package's binary with "aufstieg del (package name)"

How to make build scripts:

You can look to any build script as an example. Build.sh contains this:


srclink="(link example)"

filename="(name of package/file)"

buildinst() {

. /path/to/aufstieg/config

...(make instructions)

}


And deps.txt contains dependencies
